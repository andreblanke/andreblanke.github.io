# [blog.blanke.dev](https://blog.blanke.dev)

My personal, Jekyll-based WIP blog deployed to GitLab pages.

## Getting started

```shell
export GEM_HOME="$(gem env user_gemhome)"
export PATH="$PATH:$GEM_HOME/bin"
```

```shell
bundle exec jekyll serve --drafts
```
